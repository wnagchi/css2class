## 开始
安装

	npm i
	
![enter description here](https://z3.ax1x.com/2021/08/10/ft6bkt.png)

##### path:目标html位置 （不局限于html 小程序的wxml等其他文档类型文件都可以）
##### cssOutPath:输出的样式文件目录
##### fileName:需要生成的文件名称 （不仅限于css scss less wxss 等都可以）
##### compression:是否压缩样式
##### baseUnit:配置出现最多的--单位
##### className:class生成规则


### 配置完成之后
	npm run css2class
